How the neural network works:

The neural network was created with an input later, two hidden layers, and an output layer. The input layer contains 19 inputs of various baseball team stats. The hidden layers contain 1000 nodes and the output contains two. This resulted in over 1,000,000 trainable parameters. The input and hidden layers activation function was chosen as Relu and the output layer used Softmax. The loss function was chosen as categorical crossentropy and the ADAM optimizer was used. This configuration produced an accuracy of 95% on the testing sets. To be able to make a prediction on all seasons, two models were made. The first model contained the even seasons (i.e. 2000, 2002, etc.) as the training set and the odd seasons as the testing set and the second model vice versa.
